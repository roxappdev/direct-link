P2P File Share

One‑click direct file transfer between computers. No server storage, fully peer‑to‑peer.

How to Use

  1. Create a Share Link – Click "Generate Link" to create a unique URL. Send this link to your recipient.
  2. Wait for Connection – Once the recipient opens the link, a direct peer‑to‑peer connection will be established automatically.
  3. Send Files – After connection, select files and click "Send Selected Files". The files will be transferred directly to the other computer.
  4. Receive Files – Incoming files will appear under "Received Files" and will be saved to your downloads.

No registration, no server storage, fully secure using WebRTC.
